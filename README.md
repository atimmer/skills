# Skills

These are the skills I use. You can install them using the `skills` npm package.

This repository contains installable agent skills.

## Install

```bash
npx skills@latest add atimmer/skills
```

Install it globally:

```bash
npx skills@latest add atimmer/skills --global
```

## Skills

* **stacked-diffs**, I found that agents don't produce good stacked diffs by default. This skill is my attempt at making them behave.
